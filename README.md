# odin-recipes

Il seguente progetto riguarda la creazione di una serie di INDEX.html sul
mio computer, in modo tale da poter realizzare un website con diversi riferi
menti a diverse pagine e immagini come è stato fatto vedere in precedenza.
QUesto sarà poi la base per utilizzare CSS e migliorare l'aspetto del sito.

Competenze dimostrate:
- ricordare i diversi passaggi per creare cartelle e pagine
- utilizzo dei comandi sul terminale per gestire velocemente il software
- utilizzo dei comandi su VSCode per la creazione di un sito
- utilizzo consapevole del linguaggio HTML per la realizzazione di un 
website
- utilizzo consapevole del commit per tenere traccia dei cambiamenti e delle 
modifiche
